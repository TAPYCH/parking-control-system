Справочник интегратора

🔌 Настройка реле разных брендов
Dingtian: Использует индексы от 0. Каналы в программе пишем реальные (1-8), ядро само переведет. URL для датчиков в Dingtian:
/api/v1/sensor/trigger?gate_id=gate_1&sensor_id=1

Rodos (ККМ): Управляется через pwr.cgi. Датчики настраиваются во вкладке M2M.
Shelly: Управляется по HTTP. Идеально для небольших объектов.
📹 Настройка камер "Линия"
В ПО Линия создайте реакцию на "Распознавание номера", выберите метод GET и вставьте URL:
http://АДРЕС:8000/api/v1/camera/event?plate=%PLATE_NUMBER%&direction=in&gate_id=gate_1
📞 Интеграция Телефонии (GSM)
АТС (Asterisk / FreePBX) должна дергать этот URL при входящем звонке на номер ворот:
http://АДРЕС:8000/api/v1/gsm/auth?phone=${CALLERID(num)}
