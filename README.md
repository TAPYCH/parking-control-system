## ⚡ О проекте

**Smart Parking OS** — это модульная платформа, которая превращает обычный шлагбаум в интеллектуальную систему. Она не зависит от вендора оборудования и позволяет управлять доступом по сложным сценариям без участия оператора.

> **Ключевая особенность:** Полная аппаратная независимость. Вы можете менять реле, камеры и датчики — логика ядра останется неизменной.

## 🚀 Возможности

### 🧠 Интеллектуальное ядро
| Функция | Описание |
| :--- | :--- |
| **Smart Reverse** | Логика реверсивных проездов с защитой от столкновений (датчики 1-2-3). |
| **Anti-Tailgate** | Защита от «паровозиков» (блокировка при попытке проезда двух авто подряд). |
| **Dynamic Limits** | **Правило 80%:** Автоматическое сокращение времени бесплатной парковки при высокой загрузке. |
| **Auto-Ban** | Автоматическое внесение нарушителей регламента времени в Черный список. |

### 🔌 Поддержка оборудования
Система работает с широким спектром IoT-устройств «из коробки»:

*   **Контроллеры:** Dingtian, Rodos (Laurent), Kernel, Shelly, Sonoff (DIY).
*   **Видеоаналитика:** Нативная интеграция с ПО "Линия" (Devline) и LPR-камерами.
*   **Светофоры:** Управление светофорным регулированием (Красный/Зеленый).
*   **Датчики:** Поддержка ИК-барьеров и индукционных петель.
*   **Пульты:** Чтение радиопультов через USB-COM конвертеры (протокол DRX).

## 💻 Интерфейс администратора

Система оснащена современным Web-интерфейсом для инженеров и операторов.

*   📊 **Live Dashboard:** Мониторинг проездов, состояния датчиков и занятости мест в реальном времени.
*   ⚙️ **Visual Config:** Настройка IP-адресов и таймингов без редактирования кода.
*   🗂️ **Database Manager:** Загрузка списков доступа из Excel/CSV.

## 🛠 Установка

### Предварительные требования
*   Python 3.9 или выше
*   Сетевое подключение к оборудованию

### Быстрый старт

```bash
# 1. Клонирование репозитория
git clone https://github.com/your-username/smart-parking-os.git
cd smart-parking-os

# 2. Установка зависимостей
pip install -r requirements.txt

# 3. Запуск ядра
python main.py
